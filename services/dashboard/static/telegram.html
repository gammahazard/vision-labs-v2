📲 Connect Telegram

Telegram turns your phone into a remote for the whole stack. From anywhere with data:

  • Get instant snapshots + AI scene descriptions when something is detected
  • Pull a live photo from any camera: /snapshot
  • Grab a 5-40s video clip: /clip 15 driveway
  • Ask the AI assistant: /ask was anyone in the backyard tonight?
  • Check who's currently in frame (/who), arm/disarm alerts (/arm)

1. Create your bot

  1. Open Telegram (mobile or desktop)
  2. Search for @BotFather and start a chat
  3. Send /newbot
  4. Pick a name and username when prompted
  5. BotFather replies with a token like 123456:ABC... — paste it below

2. Send a message to your bot

Open Telegram and send any message (or just /start) to your bot: @yourbot

We'll wait up to 30 seconds, then capture your chat ID so the bot knows where to send alerts.

⏳ Waiting for your message…

3. All set

Connected as …. A test message has been sent to your Telegram — check your phone to confirm. The dashboard is restarting (~10 seconds) so the bot's command poller picks up the new token.

🤖 Bot connection Configured

To swap to a different bot or re-pair from a different Telegram account, run the connection flow again — the new token replaces the current one on save.
