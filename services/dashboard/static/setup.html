Welcome.

Vision Labs is a multi-camera AI security stack — person and vehicle detection, face recognition, DVR recording, and a chat assistant that can answer questions about what's happening across your cameras.

This wizard will:

  • Detect your GPU and recommend a configuration
  • Help you add your first camera
  • Drop you into the dashboard

You can re-run this wizard later by deleting /data/setup-state/setup.json inside the dashboard container's auth-data volume.

Detect your hardware.

We'll ask the orchestrator to spawn a one-shot nvidia-smi probe so we can see what GPUs are available. This may take up to ~30 seconds on the first run while the probe image downloads.

Add your first camera.

You can scan your network for ONVIF-compatible cameras (Reolink, Hikvision, Dahua, etc.), or enter an RTSP URL manually. Skip the step to add cameras later from the Cameras tab.

Connect to camera

Enter the camera's credentials. We'll use them once to fetch the RTSP URL via ONVIF.

You're done.

From here you can:

  • Open the live grid to see your camera feeds
  • Enroll faces in the Settings tab so identities show on alerts
  • Add more cameras any time via the Cameras tab
  • Configure Telegram notifications in your .env file
